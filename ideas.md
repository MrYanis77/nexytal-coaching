<text>
**Design Movement** : Minimalisme Institutionnel Premium — inspiré des grands cabinets de conseil (McKinsey, BCG, Acteo, Talentis)

**Core Principles** :
1. Espace blanc comme élément actif — respiration maximale entre les sections
2. Hiérarchie typographique forte — titres serif imposants vs corps sans-serif épuré
3. Palette restreinte et intentionnelle — bleu nuit / or sable / vert sauge
4. Animations discrètes et purposeful — rien de gratuit, tout renforce la confiance

**Color Philosophy** :
- Bleu nuit #0A2540 : autorité, confiance, profondeur — couleur des institutions
- Or sable #C9A24B : excellence, premium, distinction — accent sparingly used
- Vert sauge #6B9080 : équilibre, croissance, humanité — contrepoint chaleureux
- Fond #FAFAF7 : blanc cassé légèrement chaud — évite la froideur clinique
- Texte #1A1A1A : quasi-noir pour lisibilité maximale

**Layout Paradigm** :
- Asymétrie calculée : texte à gauche, visuel à droite sur hero
- Sections alternant fond blanc et fond bleu nuit pour rythmer la lecture
- Grilles 12 colonnes avec gouttières généreuses
- Sections pleine largeur pour les chiffres clés et CTA

**Signature Elements** :
1. Ligne dorée fine (#C9A24B) comme séparateur ou accent de titre
2. Cartes avec bordure gauche dorée (4px) pour les témoignages et citations
3. Numéros de section en grand format transparent en arrière-plan

**Interaction Philosophy** :
- Hover sur cartes : légère élévation + bordure dorée apparaît
- Boutons CTA : scale(0.97) au clic, transition 160ms ease-out
- Navigation : mega-menu avec animation fade + slide subtile

**Animation** :
- Entrée des sections : fade-in + translateY(20px) → 0, 600ms ease-out, stagger 80ms
- Compteurs animés sur les chiffres clés
- Carousel témoignages : glissement horizontal fluide

**Typography System** :
- Titres H1/H2 : Fraunces (serif, italic possible pour emphase), 600-700 weight
- Corps : Inter, 400/500, line-height 1.7
- Labels/badges : Inter 600, uppercase, letter-spacing 0.1em

**Core Principles** :
1. Grille visible et assumée
2. Typographie massive et contrastée
3. Couleurs franches sans dégradés
4. Interactions directes et sans ambiguïté

**Color Philosophy** :
- Blanc pur #FFFFFF fond
- Bleu électrique #1B4FFF pour les accents
- Noir #000000 pour les textes et bordures
- Jaune #FFD700 pour les highlights

**Layout Paradigm** :
- Blocs rectangulaires avec bordures visibles
- Pas d'arrondis — angles droits assumés
- Sections délimitées par des lignes épaisses

**Signature Elements** :
1. Bordures noires épaisses (2-3px) sur tous les éléments
2. Ombres portées décalées (offset shadows)
3. Texte oversized qui déborde des conteneurs

**Interaction Philosophy** :
- Hover : changement de couleur de fond instantané
- Pas d'animations — réactivité immédiate
- Micro-interactions directes

**Animation** :
- Transitions instantanées ou très courtes (100ms)
- Pas de fade — apparition directe
- Déplacement physique des éléments au hover

**Typography System** :
- Titres : Space Grotesk Bold
- Corps : DM Mono
- Labels : Space Grotesk uppercase
